◆ creer_serveur()

int creer_serveur ( int  port)

Crée une socket serveur qui écoute sur toute les interfaces IPv4 de la machine sur le port pass é en paramètre. La socket retournée doit pouvoir être utilisée directement par un appel à accept. La fonction retourne -1 en cas d’erreur ou le descripteur de la socket créée.

On met en place le serveur dans cette fonction en précisant le port que l'on souhaite écouter.

Paramètres
portLe port sur lequel on souhaite que le serveur écoute.
Renvoie
Le socket du serveur.
